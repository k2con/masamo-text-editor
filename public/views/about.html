Masamo Text Editor

A simple and easy to use WYSIWYG editor built with JavaScript.

Getting statrted

Get statrted by including our production ready CSS and JavaScript files in your HTML file.

1. Create a new HTML file. 

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="utf-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1">
                        <title>Masamo Text Editor</title>
                    </head>
                    <body>
                        <div id="masamo-editor">
                            <h2 class="text-center">Masamo Text Editor</h2>
                            <p class="lead text-center">
                                A simple and easy to use WYSIWYG editor built with JavaScript.
                            </p>
                            <hr class="my-4">
                        </div>
                    </body>
                    </html>

2. Include the CSS and JavaScript files in the head and body tags respectively. 

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="utf-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1">
                        <title>Masamo Text Editor</title>
                        <link rel="stylesheet" href="<%= baseUrl %>/assets/mte/css/style.css">
                    </head>
                    <body>
                        <div id="masamo-editor">
                            <h2 class="text-center">Masamo Text Editor</h2>
                            <p class="lead text-center">
                                A simple and easy to use WYSIWYG editor built with JavaScript.
                            </p>
                            <hr class="my-4">
                        </div>
                        <script src="<%= baseUrl %>/assets/mte/js/app.js"></script>
                    </body>
                    </html>

3. Create a new JavaScript with the following code and include it in the body tag after the app.js file. 

                
                    const editor = new MasamoEditor("#masamo-editor");

Your HTML file should look like this: 

                
                    <!DOCTYPE html>
                    <html lang="en">
                    <head>
                        <meta charset="utf-8">
                        <meta name="viewport" content="width=device-width, initial-scale=1">
                        <title>Masamo Text Editor</title>
                        <link rel="stylesheet" href="<%= baseUrl %>/assets/mte/css/style.css">
                    </head>
                    <body>
                        <div id="masamo-editor">
                            <h2 class="text-center">Masamo Text Editor</h2>
                            <p class="lead text-center">
                                A simple and easy to use WYSIWYG editor built with JavaScript.
                            </p>
                            <hr class="my-4">
                        </div>
                        <script src="<%= baseUrl %>/assets/mte/js/app.js"></script>
                        <script src="https://example.com/js/index.js"></script>
                    </body>
                    </html>

4. Open the HTML file in your browser and you should see the editor.
